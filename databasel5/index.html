B+ Tree

和B Tree的区别在于：B+只有叶子存放数据，而B Tree的中间节点也存放数据。

几乎和上面一样，但是多了

  • dynamic tree index
    • always balanced
    • support efficient insertions and deletions
      • grows at root not leaves

注意:

  • 占有率：几乎半满，除了root
  • 底部DLL
  • $max\ fan\ out = 2d + 1$
