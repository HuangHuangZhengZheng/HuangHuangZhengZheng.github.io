文件状态

未跟踪-未修改-已修改-暂存 git add <name> - *->暂存 git commit -m "message" - 暂存->未修改 git rm <name> - 未修改->未跟踪

查看状态

1

git status

更加细致几行几列

1

git diff

查看历史日志

1
2
